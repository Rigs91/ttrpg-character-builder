Start It

Expected Launch Path

Double-click START_FORGE_CHARACTER_AI.cmd for the intended Windows flow. It starts the local stack, then opens the guided builder at index.html.

One-click Windows launcher ./START_FORGE_CHARACTER_AI.cmd
PowerShell launcher npm.cmd run start:ai
Manual dev stack npm.cmd run dev
  • The main builder is the guided root UI at index.html.
  • If the API is healthy but AI is unavailable, the chat falls back to guide mode until Ollama is ready.
  • The React app at http://localhost:5173 still exists as a secondary architecture path.

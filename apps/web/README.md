# @forge/web

React + Vite client for the Forge Character MVP.

## Local Development

From the repo root:

```bash
npm.cmd run dev
```

Or from this workspace:

```bash
npm.cmd run dev
```

Default endpoints:

- Web UI: `http://localhost:5173`
- API base: `http://localhost:8787`

Override the API target with `VITE_API_BASE_URL`.

Use the root README for the full monorepo workflow and QC commands.

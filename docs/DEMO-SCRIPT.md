# 60-Second Demo Script

## Goal

Show that Forge Character turns a rules-heavy process into a guided, understandable workflow.

## Script

**0:00 - 0:08**

"This is Forge Character, a guided tabletop RPG character builder. The main idea is to make character creation feel structured instead of overwhelming."

**0:08 - 0:18**

"I’ll start in Setup. Here I can choose the ruleset, set the character concept, and lock in things like class, species, background, and level."

**0:18 - 0:30**

"When I move into Ability Scores, the builder immediately shows how choices affect the character and flags issues as I go. The goal is to keep the rules visible without dumping everything on one screen."

**0:30 - 0:42**

"In Build, I can choose skills, feats, items, and spells. The live sheet on the right keeps the current state grounded while I’m making decisions."

**0:42 - 0:53**

"On Review, I can validate the build, save locally, export JSON, or generate a parsable PDF. The same repo also includes a React/API stack and an optional local Ollama assistant for free-text autofill."

**0:53 - 1:00**

"What I like about this project is that it’s not just a rules engine. It’s a workflow design exercise that balances flexibility, guardrails, and speed."
